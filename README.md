# QuickBite

QuickBite is an online food ordering system developed using PHP and MySQL. The platform allows users to browse menu items, place orders, and manage their accounts through a simple and responsive interface.

## Features

* User Registration and Login
* Food Menu Browsing
* Online Food Ordering
* Order Management
* Admin Panel
* Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap
* PHP
* MySQL
